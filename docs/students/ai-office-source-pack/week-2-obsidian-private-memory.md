# Week 2: Obsidian Is Private Memory

Recorded: 2026-06-11
Audience: GitMoneyOS students
Layer classification: GitHub record

## Lesson Objective

Use Obsidian as the staging ground before work deserves GitHub.

Obsidian is where your raw thinking can be unfinished. The point is not to make pretty notes. The point is to capture real material and shape it into something an agent or official record can use.

## Worked Example

Capture note:

```text
Title: Pricing objection I keep answering
Owner: me
Source: sales call
Status: seed
Classification: private
Summary: I answer the same pricing objection every week.
promotion_target: sop
```

Next move:

```text
Turn this note into an agent brief for drafting a pricing objection SOP.
```

## Proof Artifact

Create one Obsidian note with:

- owner,
- source,
- status,
- classification,
- summary,
- `promotion_target`,
- `created`.

## Promotion Boundary

Private notes stay private until you deliberately promote them. Promotion is a decision, not a default.
