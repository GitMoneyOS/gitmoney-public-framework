# Acceptance Rubric

How the owner (or an agent pre-reviewing for the owner) judges whether work is done. Used at Stage 5 of the workbench contract and at cohort checkpoints.

## Judging An Agent Artifact

Score each line pass/fail. Any fail means revise, not done.

| Check | Question |
|---|---|
| Brief fidelity | Does the artifact deliver exactly what the brief's done-when describes? |
| Scope honesty | Did the work stay inside scope; are out-of-scope discoveries listed, not done? |
| Evidence | Is proof attached that the artifact works or is complete? |
| Draft discipline | Is it presented as a draft awaiting approval, not as a finished fact? |
| Boundary respect | Were the hard rules untouched - no secrets, no publishing, no permission changes? |
| Traceability | Can you walk artifact -> brief -> source note without help? |

## Judging A Promotion Candidate

The asset test, plus:

| Check | Question |
|---|---|
| Classification | Deliberately set by the owner, not defaulted? |
| Safety check | Run, with zero unresolved flags? |
| Target fit | Is the artifact formatted for its target per `promotion-rules.md`? |
| Standalone | Does it make sense to a reader with no access to the vault? |

## Judging Student Progress (Week 2 Checkpoint)

A student passes when they can demonstrate, live:

1. Explain the three layers: Obsidian is private memory, agent platforms are workbenches, GitHub is the official record.
2. Show one note with owner, source, status, classification, summary, and promotion target filled.
3. Show one agent brief created from one of their own notes.
4. Show one work item that moved across the Workbench Board.
5. Name one asset candidate from their own notes, prompts, workflows, content, or business knowledge.
6. Say, for one of their notes, what stays private and what becomes a GitHub issue, repo doc, decision log entry, content asset, SOP, or agent brief - and why.

No live GitHub access is required to pass. A queued promotion with a complete artifact counts.

## The Tone Of Review

Review is a gate, not a judgment of the person. "Not yet, here is what is missing" is a complete and kind review. The rubric exists so nobody has to argue from feelings.
