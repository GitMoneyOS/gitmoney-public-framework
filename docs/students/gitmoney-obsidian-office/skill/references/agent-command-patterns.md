# Agent Command Patterns

## Purpose

Define common command patterns for agents working in the vault.

## Owner

Red Pillar and Hitsuyo Aku, under KnowTheLedge.

Copyable commands for students, and the behavior the agent owes for each. When the user gives a vague job, route them to the nearest pattern instead of guessing.

## Capture And Shape

> "Capture this: [idea]"

Create an inbox note from the daily capture template, fill the six student-facing frontmatter fields plus `created`, ask only for what cannot be inferred.

> "Process my inbox"

Walk `00-inbox/` oldest-first. For each note: propose a lane, a status, and one real link. The user approves per note or in bulk; never file silently.

> "Link this note"

Suggest up to three "This reminds me of..." connections with one sentence of justification each. Quality over count.

## Route And Work

> "Turn this note into an agent brief"

Draft a brief from `AGENT_BRIEF_TEMPLATE.md`: job statement, scope in/out, deliverable, boundaries, done-when. Show it for approval, then add it to the board's Backlog.

> "Work the board"

Read `WORKBENCH_BOARD.md`. State what is in Doing. If under the WIP limit of 3, pull the top Ready item and execute its brief. Move cards with dates as you go.

> "What's blocked and why?"

List Blocked items with what each is waiting on, plus anything in Review aging past a week.

## Audit And Mature

> "Run a vault health audit"

Stage 1 of `knowledge-to-asset-pipeline.md`. Numbers plus the three worst findings, no padding.

> "What patterns do you see in my notes?"

Stage 2: repetition with at least two linked source notes per pattern. Hunches labeled as hunches.

> "Score this note as an asset"

Stage 3: the five-criterion score with what is missing, and the honest verdict - including "this is a sprout, leave it alone."

## Promote

> "Prepare this for promotion"

Run `promotion-rules.md`: confirm asset test, propose the target, produce the artifact, run the public-safety check, queue it in `OWNER_DECISION_QUEUE.md`. Never push, publish, or post.

## Review

> "Run my weekly review"

The weekly review template, driven: inbox to zero, board walked, stale seeds archived, one summary note. Twenty minutes, not two hours.

## When The Job Is Vague

"Make my vault better" gets this response: "Which of these do you want - inbox processing, a health audit, pattern surfacing, or a brief for a specific job?" One clarifying question, then work. Never invent scope.
